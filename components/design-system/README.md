# Local Design-System Source

This directory is scaffolded by `@nikolayvalev/design-system` CLI.

Install flow:
1. Configure your MCP client to use `https://designsystem.nikolayvalev.com/mcp`.
2. Call MCP tools like `get_component_bundle`, `get_section_bundle`, and `get_page_bundle`.
3. Write returned files inside `components/design-system`.
4. Commit generated source files to your repository.

Token profile selected during init: `public`
Recommended CSS import:
`@nikolayvalev/design-tokens/styles/public.css`
